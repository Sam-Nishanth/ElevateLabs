\# Titanic Dataset - Exploratory Data Analysis



\## Objective

Perform Exploratory Data Analysis (EDA) on the Titanic dataset using statistical summaries and visualizations to identify relationships, trends, and patterns.



\## Tools Used

\- Python

\- Pandas

\- Matplotlib

\- Seaborn

\- Plotly



\## Analysis Performed

\- Dataset Overview

\- Summary Statistics

\- Missing Value Analysis

\- Histograms

\- Boxplots

\- Correlation Heatmap

\- Pairplot

\- Survival Analysis

\- Interactive Plotly Histogram



\## Outcome

The analysis provides insights into passenger demographics, fare distribution, survival patterns, feature relationships, and potential outliers.
