# Titanic Data Cleaning \& Preprocessing

## Objective

This project demonstrates data cleaning and preprocessing techniques using the Titanic dataset for Machine Learning.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Steps Performed

1. Loaded the Titanic dataset.
2. Explored dataset information.
3. Checked for missing values.
4. Filled missing values using:

   * Median (Age)
   * Mode (Embarked)
5. Dropped the Cabin column due to many missing values.
6. Converted categorical variables into numerical values using Label Encoding.
7. Standardized numerical features using StandardScaler.
8. Visualized outliers using boxplots.
9. Removed Fare outliers using the IQR method.

## Output

The dataset is cleaned and preprocessed, making it suitable for machine learning models.
